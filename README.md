# NoiseModelling Covadis

This repository host the Covadis Platform, this is a strategic noise maps generator using NoiseModelling computation core and Covadis environemental database specification French standard.

The goal is to make a web platform able to quickly generate strategic noise maps on the country level scale using input/output data following accurate specification.

This platform may use High Power Computing platform (Slurm) in order to run the most computing intensive tasks.
